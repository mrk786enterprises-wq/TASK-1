# 🌐 Personal Portfolio Website

## 📖 Description

This is a personal portfolio website developed using HTML5 and CSS3 as part of the DecodeLabs Frontend Development Internship (Week 1 Project).

The website presents my personal profile, education, technical skills, projects, and contact information through a clean, responsive, and user-friendly interface.

---

## ✨ Features

- Responsive Navigation Bar
- Hero Section
- About Me Section
- Skills Section
- Projects Section
- Contact Section
- Footer
- Clean UI Design
- Responsive Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
